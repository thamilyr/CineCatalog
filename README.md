## 📌 Sobre o projeto

O **CineCatalog** é uma aplicação de console desenvolvida em Java que permite ao usuário:

* cadastrar filmes com informações detalhadas,
* listar todos os filmes cadastrados,
* buscar filmes por título ou por categoria,
* armazenar e recuperar dados localmente.

Este projeto foi criado com foco em aplicar conceitos de Programação Orientada a Objetos (POO), organização de código e boas práticas de desenvolvimento.

---

## 🧰 Tecnologias utilizadas

* **Java 17+**
* **Maven** (gerenciamento de dependências)
* Programação Orientada a Objetos
* Leitura e escrita de arquivos (persistência simples)

---

## 🚀 Funcionalidades

✔ Cadastrar filmes com título, gênero e ano
✔ Listar todos os filmes cadastrados
✔ Buscar filmes por título
✔ Buscar filmes por gênero
✔ Persistência de dados entre execuções

## ▶️ Como executar o projeto

### 1. Clone o repositório

```bash id="clone_cinecat"
git clone https://github.com/thamilyr/CineCatalog.git
```

### 2. Acesse a pasta do projeto

```bash id="cd_cinecat"
cd CineCatalog
```

### 3. Compile e execute

```bash id="build_cinecat"
mvn clean install
mvn exec:java
```
